# NYCO Construction — Brand color palette

## Primary (blues from logo)

| Name | Hex | Usage |
|------|-----|-------|
| Navy | `#1B3A6B` | Headings, navbar, footer |
| Royal blue | `#1E4D8C` | Buttons, CTAs, links |
| Steel blue | `#2A6AB5` | Secondary buttons, icons, hover states |

## Accent (orange)

| Name | Hex | Usage |
|------|-----|-------|
| Construction orange | `#E8731A` | Primary accent, highlights, CTAs |
| Light orange | `#F5A85C` | Hover states, badges, secondary accent |
| Deep orange | `#C45E10` | Active/pressed states, dark accent |

## Neutrals

| Name | Hex | Usage |
|------|-----|-------|
| White | `#FFFFFF` | Page background |
| Off-white | `#F5F6F8` | Section backgrounds, cards |
| Silver | `#D3D8E0` | Borders, dividers |
| Cool gray | `#B0B8C4` | Muted text, placeholders |
| Charcoal | `#2C3340` | Body text, dark backgrounds |

## CSS variables

```css
:root {
  --color-navy: #1B3A6B;
  --color-royal-blue: #1E4D8C;
  --color-steel-blue: #2A6AB5;
  --color-orange: #E8731A;
  --color-orange-light: #F5A85C;
  --color-orange-dark: #C45E10;
  --color-white: #FFFFFF;
  --color-off-white: #F5F6F8;
  --color-silver: #D3D8E0;
  --color-cool-gray: #B0B8C4;
  --color-charcoal: #2C3340;
}
```
